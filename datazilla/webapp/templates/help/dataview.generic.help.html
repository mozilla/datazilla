View Navigation

Each data view has a navigation menu displayed in the top left panel. When a new navigation position is chosen the associated view will initialize accordingly. If a data collection is selected the browser page will be cleared of all open views and replaced with the view collection that was selected.

Control Panel

The control panel is composed of a form with fields that pertain to the view displayed. The form values can be modified and the view reloaded by selecting the load button at the bottom of the control panel. The control panel form values are only applied to the associated view, with the exception of the date range.

The date range is passed on as signal data to any child view, so you can control the date range of a set of child views by making a single modification in the parent's control panel.

To look for similiarity in a particular field, first edit the field to contain the target text then insert a '%' at the beginning, end, or both beginning and end of the field data.

Parent/Child Relationships

Data views in datazilla have a parent/child relationship that is indicated in the top panel of each view. A view's parent can send signals to its child views when the user selects links or graphical representations of data that can send signals. The parent/child relationship of any view is identified by a data view number "View:Some Number" that refers to the data view selected and its parent "Parent:Some Page Name, Some View Name". Anytime a new view is opened from an existing view, that view becomes the parent. Parents can send signals to child views that exist on different browser tabs or pages and there is no limit to the number of child views a parent can have.

Visualization

View data can be visualized in tabular or graphical form. Different visualizations of the data from a view can be accessed using the Visualization menu, some views will only have a tabular presentation.

Button Controls

Each view has a set of button controls displayed in the top panel. The following is a description of each button from left to right.

Decrease View Panel Size: Decrease the display size of a view's data table. This may or may not apply to a graphical representation.

Increase View Panel Size: Increase the display size of a view's data table. This may or may not apply to a graphical representation.

Open New View: This button opens a modal window that presents the user with options for opening new child views. The child views opened will be parented to the view from which they were opened.

Help: Clearly you understand this button...

Refresh: Reload the data from its original source in the view.

Move To New Window: Move the view data to a new browser tab or window.

Close: Close the view.

Signals

Signals can be sent from a parent view to its children by selecting links displayed in tabular data or graphical representations of data that can send signals. When a signal is sent, it's displayed in the parent view in the middle panel and prefixed with "Sent:", when a signal is received by a child view it's displayed in the parent view in the middle panel and prefixed with "Received:". A date range is always displayed in the signal display panel and is prefixed with "Range:". The date range can be set by opening the control panel and modifying the date range displayed.

Table Filters

Directly above each tabular display of data is a set of controls that operate on the table. The controls are described below from left to right.

Show Entries: This option menu can be used to changed the number of rows displayed on a single page. When the number of rows exceeds the page display limit a set of paging links are displayed in the lower right of the table.

Column Filter: This option menu can be used to change which columns are displayed. When opened, all of the columns available in a table are displayed along with a checkbox that can be used to toggle the display of a column.

Search Filter: The search filter will filter all of the data contained within a data table by any string or regular expression entered into the input box.
